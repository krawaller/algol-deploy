Info on Murus Gallicus

Murus Gallicus is probably the most famous game by prolific designer Philip Leduc, who also designed Momentum and Go with the floe. And for good reason - it is a gem of a game, and one of our absolute favourites!

The Nestor Games editionPhoto by Nestor Games

One of the cooler features is how material equality between the players is maintained throughout the battle, so everything boils down to positioning which makes for a very strategic game.

Murus Gallicus is a stacking game with Chessicals doesn't support, so we use different unit types to represent various stack heights (just like in Descent).

Chessicals also implements the advanced variant with catapults, which Philip Leduc described in a BGG thread.
